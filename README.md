# GHRC

GHRC is a tool for managing **G**it**H**ub **R**epository **C**onfigurations in a declarative way.

Repository configuration includes

- Visibility
- Labels
- Topics
- Protected Branches
- Description

## Environment Variables

- `GHRC_GITHUB_TOKEN`
- `GHRC_GITHUB_API` (default: "https://api.github.com")
